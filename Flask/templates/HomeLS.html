Location Detection

This innovative web application is designed to seamlessly detect the location of an Indian city within an uploaded image. Leveraging the power of deep learning, this cutting-edge model has been trained on a diverse dataset comprising 10,500 images, meticulously categorized into five distinct classes: Ahmedabad, Delhi, Kerala, Kolkata, and Mumbai. Developed in collaboration with Parul University, the model has attained a commendable test accuracy of 55.6%.

Using the App is a Breeze:

  • This method is to run app on Streamlit.
  • Clone the Repository: Begin by cloning the GitHub repository onto your system using the command provided.
    git clone https://github.com/shahdivax/Location-Classification-of-Indian-Cities.git
  • Install Dependencies: Ensure the necessary libraries are in place by executing the prescribed 'pip install' command.
    pip install -r requirements.txt
  • Run the App: Launch the app via the 'streamlit run' command, following the instructions to switch to the 'master' branch.
    streamlit run app.py
    Make sure to switch to the master branch before running the app.
  • Image Upload: Upload a JPG or JPEG image, preferably one with a clear view of the city's landmarks.
  • Instant Prediction: Sit back and watch as the app processes your image, accurately predicts the city's location, and displays the predicted location along with the achieved accuracy percentage.

It's worth noting that for optimal results, the app is most effective when presented with images that offer a distinct view of the city's landmarks. Images that lack clarity or don't provide a clear representation might not yield accurate outcomes.

For those curious, a live demonstration of the app's capabilities is readily available here, providing a hands-on experience of its functionalities.

Under the hood, this project is underpinned by robust Python coding. Key libraries employed include Flask, which serves as the framework for constructing the user interface, and TensorFlow with Keras, harnessed to load the pre-trained model and process images effectively. Numpy and Pillow make appearances for seamless Data and Image processing, respectively.

The app's inner workings follow a logical sequence:

  1. Model Loading: The trained deep learning model is loaded into the app, setting the stage for its predictive prowess.
  2. City Labels: Clear labels for the five Indian cities are defined, forming the foundation for accurate predictions.
  3. Accuracy Threshold: An intelligent minimum accuracy threshold is established to ensure reliable predictions.
  4. Image Processing: A dedicated function is designed to efficiently process the images uploaded by users.
  5. User Interface: The Streamlit app interface presents itself, complete with a user-friendly file uploader.
  6. Prediction and Display: Uploaded images undergo processing, leading to the display of predicted location and corresponding accuracy.
